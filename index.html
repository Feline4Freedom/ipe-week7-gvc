Global Value Chains — Workshop Activities

From Efficiency to Resilience. Two interactive workshops, ~50 minutes each. Open in any modern browser; no installation required.
STAGE A · 50 MIN

Supply Chain Risk Assessment

Diagnostic workshop · group of 4–5 · output: risk matrix + infographic

Pick one of six multinationals. Map its supply chain. Fill an interactive risk matrix. Identify single-points-of-failure. Generate an executive summary that updates live as you fill the blanks.

  • 6 companies spanning semis (TSMC, ASML, Foxconn) and minerals (BYD, MP, Glencore)
  • Live SPOF counter + high-risk node tracker
  • Auto-updating executive summary with fill-in-the-blank scaffold
  • Print / Save as PDF for portfolio submission

For Instructors — How to Distribute

Both files are fully self-contained (single HTML file each, no internet required after first open). To run a session:

  • Activity A — share the same URL with all groups; assign a different company to each group at the start.
  • Activity B — share the public URL Week7_Activity_B_Negotiation_EN.html with everyone, then privately give each delegation its 4-character code. Alternatively, send pre-filled URLs like ...?role=US&code=PCM7 directly to the assigned group's group-chat.
  • Teacher view — keep ...?teacher=1 on your laptop only. Never project this to the room.
  • Delegation codes: US=PCM7 · JP=KIKU · EU=AURA · KR=HANL · VN-IN=BHAR · AU=BRAV
