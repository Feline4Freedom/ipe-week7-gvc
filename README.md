# IPE Week 7 — Global Value Chains

Interactive workshops for Week 7 of **International Political Economy (IMAPA / National Sun Yat-sen University, Fall 2025)**: *Global Value Chains — From Efficiency to Resilience*.

🌐 **Live site**: <https://feline4freedom.github.io/ipe-week7-gvc/>

## What's here

| File | What it does |
|------|--------------|
| [`index.html`](./index.html) | Landing page — picks between Activity A and B |
| [`Week7_Activity_A_RiskAssessment_EN.html`](./Week7_Activity_A_RiskAssessment_EN.html) | **Activity A** — Supply-chain risk assessment workshop. Pick one of 6 multinationals (TSMC · ASML · Foxconn · BYD · MP Materials · Glencore), map its chokepoints, fill an interactive risk matrix, identify single-points-of-failure. |
| [`Week7_Activity_B_Negotiation_EN.html`](./Week7_Activity_B_Negotiation_EN.html) | **Activity B** — Pacific Critical Minerals Compact mock negotiation. Six delegations (US · JP · EU · KR · VN-IN · AU) negotiate friend-shoring rules; China observes. Each delegation's CLASSIFIED dossier is unlocked by a 4-letter code. |

## How to use in class

1. Share the live URL above with the whole class.
2. **Activity A** (50 min): assign one company per group; students click through and fill the risk matrix in-browser.
3. **Activity B** (50 min): give each group its delegation code (US=PCM7 · JP=KIKU · EU=AURA · KR=HANL · VN-IN=BHAR · AU=BRAV). Codes unlock the role's confidential dossier. Instructor accesses the **god's-eye view** at `Week7_Activity_B_Negotiation_EN.html?teacher=1` (which includes China observer scripts and the conflict prediction matrix — *do not project this to students*).
4. After both activities, students click **📝 Submit Feedback** to fill the linked Google Form. Submissions feed into the instructor's Sheet.

## Tech notes

Pure HTML / CSS / vanilla JavaScript. No build step, no external dependencies, no analytics. Works fully offline once loaded.

Designed to fit a 16:9 classroom projector but responsive down to mobile.
